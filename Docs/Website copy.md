# What Is CloaKey?

CloaKey is a lightweight workflow protection utility.

It creates a temporary protection layer between your keyboard, mouse, and operating system.

Applications continue running normally.

Only interaction changes.

Think of it as:

> A seatbelt for active workflows.

# FAQ

### Does CloaKey stop applications?

No.

Applications continue running normally.

Only interaction is affected.

---

### Does CloaKey record keystrokes?

No.

Input is never stored or transmitted.

---

### What happens if I forget the unlock shortcut?

Timed locks can automatically restore access.

Emergency recovery is always available.

### Can I use it while AI agents are running?
